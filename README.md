# Asteroids
Asteroids game java clone.

### Execution
In the project root directory:
```
./gradlew run
```

---
Alternatively, you can create a jar file.
```
gradle build
```
And run with:
```
java -jar build/libs/Asteroids-1.0.jar
```

### Testing
In the project root directory:
```
./gradlew test
```
